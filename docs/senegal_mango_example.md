# Senegal Example: Mango Drying

## Scenario
Cooperative: Casamance Mango Cooperative (Ziguinchor)

## Input
- Fresh mango input: `1000 kg`

## Process
- Washing -> Sorting -> Slicing -> Drying (`16h`) -> Packaging

## Output
- Dried mango output: `780 kg`

## Loss Calculation
- Loss (kg): `1000 - 780 = 220 kg`
- Loss (%): `(220 / 1000) * 100 = 22.0%`

## Interpretation
- 22% loss is above threshold (20%), flagged as anomaly.
- Recommended actions:
  - Improve pre-drying sorting quality controls.
  - Add humidity checks every 2 hours.
  - Rotate trays to reduce uneven drying.
